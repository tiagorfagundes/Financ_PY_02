# Camada 2 — Regulatória Operativa (Operação de Financiamento via Dívida Pública PY)

Camada operativa para a operação definida com a empresa: **financiamento envolvendo títulos de dívida pública do Paraguai**, em mercado primário e secundário, com **monetização legítima** (repo/colateralização de títulos reais), e **financiamento de consórcio/PPP**, podendo a empresa atuar em **três papéis** conforme o projeto.

> Esta camada desce da arquitetura (Camada 1) para o **funcionamento**: leis-quadro + decretos + resoluções operativas + gates de licença. Próximas camadas: 3 (mapa transação→norma), 4 (matriz de viabilidade), 5 (gap analysis + reforma).

---

## Operação travada (Camada 0 — definição)

| Item | Definição confirmada |
|------|----------------------|
| "Monetizar" | **Repo / colateralização / desconto de título real** — título existente, custodiado, precificado a mercado, usado como garantia para captar funding e repassar. **Não** inclui "monetização de instrumentos" via plataformas/leased/MTN (fora de escopo, risco de fraude/OFAC). |
| Eixo A | Investidor/dealer em dívida soberana PY (primário + secundário + repo) |
| Eixo C | Financiamento de consórcio/PPP com recebíveis de contrato público |
| Papéis | (i) **Investidora**, (ii) **Intermediária licenciada**, (iii) **Arranjadora** — construído para os três |
